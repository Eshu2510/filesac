# filesac

file-functions for node.js that return Promises

## Installation

[`npm i filesac`](https://www.npmjs.com/package/filesac)

## Import

```
const {
    textFileContent,
    writeTextInFile,
    concatenateFiles,
    copyFile,
    deleteFile
} = require("filesac");
```
